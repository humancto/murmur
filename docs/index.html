01 — Dictation, returned to you

Murmur.

Private Mac dictation. Built for accents.

A free, open-source alternative to Wispr Flow. Holds nothing in the cloud. Records nothing about you. Runs Whisper on the Apple Neural Engine, cleans up disfluencies with a small local model, and pastes into the focused field — Terminal, iTerm2, VS Code, Slack, anywhere you can type.

03 — How it works

Five steps.
Zero round-trips.

Audio · ANE · paste

01
Capture
Push-to-talk hotkey opens the mic. 16 kHz mono PCM ring buffer.
CoreAudio
02
Trim silence
Voice activity detection cuts dead air before transcription. Stops Whisper from hallucinating on quiet.
Silero VAD
03
Transcribe
Whisper large-v3-turbo runs on the Apple Neural Engine, biased by your personal vocabulary.
WhisperKit · ANE
04
Clean up
Qwen2.5-3B fixes punctuation and removes ums and uhs. Strict prompt — never rephrases your voice.
llama.cpp · XPC
05
Inject
Clipboard-paste primary; direct accessibility insertion on a curated allowlist of native apps.
Clipboard · ⌘V

05 — Privacy

Three zeros.
One promise.

Verifiable · in the source

0
Network calls at runtime
Verified with Little Snitch. The only outbound traffic is the one-time model download on first launch — and Sparkle update checks, which you can disable.
0
Telemetry events
No anonymized usage. No crash reports phoned home. Crashes are written to ~/Library/Logs/Murmur/ and stay there unless you choose to file them.
0
Account, subscription, paywall
Nothing to log into. Nothing to pay for. Microphone indicator in the menu bar whenever audio is captured. Password fields refused on principle.

08 — Built in public

Every PR.
Reviewed by an expert.

Architecture · diff · tests

Murmur is built one merged PR at a time, on a strict ROADMAP-driven loop. Each item is planned, the plan is reviewed adversarially by an apple-expert agent, the branch is implemented with atomic commits and tests, the diff goes through a second adversarial review, and only an APPROVE verdict on the actual merged diff lets it land. No exceptions, no exemptions.
